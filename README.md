# Stellenwerttafel / Place Value Chart

An app to learn the decimal place value chart inspired by the corresponding [iOS app](https://apps.apple.com/us/app/place-value-chart/id568750442). Based on the research of Ulrich Kortenkamp.

Built with the framework [libgdx](https://libgdx.com/) and Kotlin.

[![Place.webm](Place.webm)](Place.webm)

# Set up

* Just compile and run the app in Android Studio, no special sauce needed :)
